## Важное ограничение этой проверки
В этой среде часть источников показывала anti-bot/challenge страницы, поэтому не все карточки удалось раскрыть автоматически. Из-за этого в таблице есть не только прямые карточки, но и целевые страницы выдачи/групп, из которых нужно добрать финальные условия через онлайн-переписку и видеозвонок.

## Как читать таблицу `listings.md`
- `reliability`: оценка надежности канала (high/med/low + комментарий).
- `booking_protection`: есть ли защита платежа платформой.
- `noise_risk`: первичная оценка шумности (для Chaweng чаще выше).
- `checked_at`: дата последней проверки URL.

## Как действовать дальше онлайн (без офлайна)
1. Открыть `search_plan.md` и идти по дневным KPI/скорингу.
2. Отправить шаблонное сообщение по Top-15 (из `shortlist.md`) с единым списком вопросов.
3. Отсеять варианты без ответа в течение 24 часов.
4. По ответившим — запросить видео-тур и видео созвона по чек-листу.
5. Получить финальный прайс на 2 месяца: аренда + электричество + вода + депозит + уборка.
6. Сверить геометку (Google Maps pin), фото и видео (без монтажа, единым проходом).
7. Проверить условия возврата депозита и письменный договор (PDF до оплаты).
8. Платить только защищенным способом (платформа/эскроу/договор + подтверждение личности).
9. Зафиксировать выбранный финалист и резервировать сразу после видео-подтверждения.
