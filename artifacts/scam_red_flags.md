# Scam red flags и безопасные практики

## Красные флаги
- Требуют предоплату «срочно сегодня», отказываются от видеозвонка.
- Отправляют только красивые фото, но не показывают объект live.
- Отказываются дать договор до оплаты.
- Не дают точный адрес/геоточку.
- Просят оплату на личный счет третьего лица без объяснения.
- Цена слишком ниже рынка без внятной причины.
- Несостыковки по имени владельца, документам и реквизитам.
- Требуют оплату вне защищенной платформы сразу в полном объеме.

## Безопасные способы оплаты/проверки
- По возможности платить через платформы с защитой (Airbnb/Booking/Vrbo).
- Для прямой аренды: договор + подтверждение личности + проверяемые реквизиты.
- Минимизировать первый платеж: бронь/депозит только после видео и договора.
- Хранить всю переписку и подтверждения в одном канале.
- Фиксировать условия одним сообщением перед оплатой.
